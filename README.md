# Modusign API Documentation Skill

Claude Code용 **모두싸인(Modusign) API 문서 스킬**입니다.

## 설치 방법

Claude Code 설정에서 이 스킬을 추가하여 모두싸인 API 관련 질문에 대한 답변을 받을 수 있습니다.

## 포함된 문서

### API 가이드
- API 소개
- 모두싸인 공공용 API 시작하기
- Postman Collections 사용
- API References

### API 사용하기
- API KEY 발급
- API KEY로 요청하기
- Rate Limit 정책
- 에러 안내

### 서명/열람 요청
- 템플릿으로 서명 요청하기
- 템플릿으로 열람 요청하기
- 서명 내용 수정 요청하기
- 맞춤 브랜드로 서명 요청하기

### Webhook
- Webhook 설정
- Webhook event

### 정보 조회
- 문서 목록 조회
- filter, orderBy, metadata 활용
- 문서 이력 조회

### 임베디드 연동
- 임베디드 서명 요청
- 내 사이트에서 서명받기
- 임베디드 문서 열람
- 임베디드 템플릿 생성/수정
- redirectUrl 활용

### OAuth 연동
- OAuth 연동하기
- 비회원 유저 연동하기

### 레시피
- Google Sheets 연동
- MCP 활용
